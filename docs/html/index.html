Documentation de collective.abctransforms¶

Documentation of collective.abctransforms by Eric Hardy.

Installation¶

add collective.abctransforms to the eggs section in the buildout.cfg file

and the source in section [sources]:

collective.abctransforms = git git@github.com:erichardy/collective.abctransforms.git

and re-run bin/buildout.

Intro¶

collective.abctransforms is a set of transforms and transform chains for Plone. So, it is targeted to Plone Developpers, not for end users.

The main starting point is the transform from a text in ABC music notation to MIDI, SVG, etc...

This set of transforms has many Linux/Unix dependencies : see Dependencies.

Most of the transforms can be configured in the control panel : see Configuration.

Compatibility : Plone 5 (not tested for Plone 4).

This module is used in the module collective.abctune
