============
Dependencies
============

All theses dependencies can be found in Linux packages. The packages given below
are **Ubuntu 14.04 LTS** packages.


abc2midi
========

Used for : convert from ABC to MIDI. MIDI files are no longer used since
quicktime plugin for browsers don't support MIDI any more.

Ubuntu package : ``abcmidi``

abcm2ps
=======

Used for : convert ABC to PDF and SVG

Ubuntu package : ``abcm2ps``. This *compiler* is written by `Jean-Fran√ßois Moine`_.


timidity
========

Used for : convert ABC (MIDI from abc2midi) to sound files : OGG, AIFF

Ubuntu package : ``timidity``

lame
====

Used for : convert AIFF to MP3

Ubuntu package : ``lame``

ps2epsi
=======


Used for : convert PS to EPSI

Ubuntu package : ``ghostscript``


ps2pdf
======


Used for : convert Postscript (PS) to PDF

Ubuntu package : ``ghostscript``

convert
=======

Used for : convert EPSI to PNG

Ubuntu package : ``graphicsmagick-imagemagick-compat``
